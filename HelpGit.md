# Подсказка по Git

## Создание репозитория:
```sh
git init
```

## Добавление содержимого:
```sh
git add
```

## Фиксация:
```sh
git commit -m "Массаге"
```

## Вызов журнала изменений:
```sh
git log
```

## Вызов журнала изменений в одну строку:
```sh
git log --oneline
```

## Команда переключение между версиями:
```sh
git checkout
```

## Отображение всех веток
```sh
git branch
```
